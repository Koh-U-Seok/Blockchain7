회원가입, 로그인, 로그아웃 추가
- 암호화 불 필요
- 회원에 따른 게시판 목록 수정 불필요
- 로그인 화면과 로그아웃 화면이 달라야함

10.24
1. jwt를 사용하여 로그인 확인
    - 웹 페이지 상에서 jwt의 데이터를 사용하여 사용자 출력
2. 로그인 시와 비로그인 시 웹 페이지 출력 다르게
    - 게시글에 대해 사용자 정보(이름, 닉네임 등등) 출력
    - 로그인 시
        - 글을 적을 수 있어야 한다.
        - 로그아웃 버튼 출력 / 로그인한 사용자 정보 출력
        - 게시자만 수정 삭제 가능
    -로그아웃 시
        - 로그인 가능
        - 게시글 작성 불가(UI가 없음)
        - 수정 삭제 불가
- jwt 를 사용한 로그인 확인
3. 일정 시간 후 로그아웃
4. 서버 상에서 로그인 체크(정상적인 데이터를 확인해야 한다.)
    - 정규, 성진, 정현, 재훈, 재일
    - 미들웨어 활용
